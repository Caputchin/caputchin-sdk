# @caputchin/mcp

MCP (Model Context Protocol) server for managing Caputchin resources — sites, API keys, and game configuration — via AI agents and MCP-compatible tools.

## Usage

```sh
npx @caputchin/mcp
```

Or install globally and run:

```sh
npm install -g @caputchin/mcp
caputchin-mcp
```
